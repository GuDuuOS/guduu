# LOGO 素材文件夹

把 GuDuu 的 logo 图片放进这个文件夹，我会用它替换项目里所有出现 logo 的地方。

## 建议放这些（有什么放什么，不用全齐）

| 文件名 | 用途 | 建议规格 |
|---|---|---|
| `logo.png` | 主 logo（方形/图标版） | 透明背景 PNG，**正方形**，≥ 512×512 px |
| `wordmark.png`（可选） | 带文字的横版 logo | 透明背景 PNG，落地页大图用 |
| `favicon.png`（可选） | 浏览器标签小图标 | 正方形，建议 256×256 或 512×512（我也能用 logo.png 自动生成） |

> 最低只要一个**正方形透明 PNG 的 `logo.png`** 我就能搞定大部分地方。

## 会替换哪些位置
1. **落地页**（cosmac.cc）—— 顶部品牌 logo
2. **聊天应用**（app.cosmac.cc）—— 浏览器标签图标、登录页 logo、应用图标
3. 其他出现品牌图标的地方

## 放好之后
告诉我一声（"放好了"），我就：
- 把图片纳入 git → 服务器 pull → 替换到落地页和聊天应用
- 更新所有 HTML / favicon 引用
